# Project: Lean One-Pager
Dieses Projekt folgt einem KI-gestützten Code-First-Ansatz basierend auf dem lokalen webdesign Skill.

## Tech Stack
- Framework: React (Vite)
- Styling: Tailwind CSS
- Icons: Lucide React
- Deployment: Vercel

## Workflow & Guidelines
- Design: Claude agiert als Senior UI/UX Designer (Modern, Clean).
- Komponenten: Modularer Aufbau in src/components.
- Assets: Nur Unsplash-Platzhalter-URLs nutzen.
- Tools: Keine externen CLI-Tools (UI Pro/Nano Banana).

## Commands
- Install: npm install
- Dev: npm run dev
- Build: npm run build
